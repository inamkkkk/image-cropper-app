# Image Cropper App

A Flutter app that allows users to select an image from their gallery and crop it.

## Features

*   Select an image from the gallery.
*   Crop the selected image using the `image_cropper` package.
*   Display the cropped image.

## Dependencies

*   `image_picker`: For selecting images from the gallery.
*   `image_cropper`: For cropping images.
*   `provider`: For state management.
*   `path_provider`: For getting temporary directory path.
*   `path`: For path manipulation.

## Getting Started

1.  Clone the repository.
2.  Run `flutter pub get` to install dependencies.
3.  Run `flutter run` to start the app.
